# utilkit

Fast, free online calculators and browser-based utility tools for finance, marketing, business, development, design, conversions, text, food, health, and games.

utilkit is designed for quick answers: calculate a mortgage payment, compare rent vs buy, format SQL, preview Markdown, build UTM links, convert colors, scale bread recipes, run a chess clock, and more.

## Online Utility Tools

Explore the live toolkit at [utilkit.com](https://utilkit.com)

### Finance Calculators

- [Mortgage Calculator](https://utilkit.com/finance/mortgage-calculator) - estimate monthly mortgage payments with taxes, insurance, PMI, and HOA fees.
- [Rent vs Buy Calculator](https://utilkit.com/finance/rent-vs-buy-calculator) - compare renting and buying with appreciation, sale costs, rent growth, and invested savings.
- [Loan Calculator](https://utilkit.com/finance/loan-calculator) - calculate fixed-rate loan payments, total interest, payoff time, and extra payment savings.
- [Credit Card Payoff Calculator](https://utilkit.com/finance/credit-card-payoff-calculator) - compare debt snowball and debt avalanche payoff plans.

### Marketing and Business Tools

- [UTM Builder](https://utilkit.com/marketing/utm-builder) - create campaign URLs with source, medium, campaign, term, and content parameters.
- [Meeting Cost Calculator](https://utilkit.com/business/meeting-cost-calculator) - estimate meeting payroll cost from attendees, duration, hourly rates, and recurrence.

### Developer Tools

- [SQL Formatter](https://utilkit.com/developer/sql-formatter) - format SQL queries with readable indentation, keyword casing, and spacing options.
- [Markdown Preview](https://utilkit.com/developer/markdown-preview) - write Markdown and preview rendered HTML side by side.

### Design, Text, and Conversion Tools

- [Color Converter](https://utilkit.com/design/color-converter) - convert CSS colors between hex, RGB, HSL, OKLCH, and Lab formats.
- [Unit Converter](https://utilkit.com/math-conversions/unit-converter) - convert length, mass, volume, temperature, time, digital storage, and more.
- [List Diff Checker](https://utilkit.com/text/list-diff-checker) - compare two lists to find common entries, missing values, and differences.

### Food, Health, and Game Tools

- [Bread Calculator](https://utilkit.com/food/bread-calculator) - scale bread recipes with baker's percentages, hydration, dough weight, and ingredient weights.
- [BMI Calculator](https://utilkit.com/health/bmi-calculator) - calculate BMI with standard or metric measurements and category guidance.
- [Multiplayer Chess Clock](https://utilkit.com/games/multiplayer-chess-clock) - run a turn timer for two or more players.
- [Spin the Wheel](https://utilkit.com/games/spin-the-wheel) - make a random picker wheel for names, giveaways, teams, dinner ideas, and quick decisions.

## Why utilkit?

- **Fast answers:** Tools are built for common everyday calculations and formatting tasks.
- **Browser-based:** No account is required for the public utility tools.
- **Search-friendly pages:** Each utility has its own URL, title, description, and category.
- **Practical categories:** Finance calculators, developer utilities, marketing tools, design helpers, text tools, food calculators, health calculators, and simple game tools live in one place.
